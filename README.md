# CreatiBot by Creative Prompt

CreatiBot is an LangChain LLM Chatbot using llama-3.1-8b-instant model, lorem ipsum dolor sit amet

## How To Install

1. Install all the modules
```bash
npm install
```
2. Create environment variables
```bash
cp .env.example .env
nano .env
```
3. Run the project
```bash
npm start
# or
node server.js
```
